HOW TO PLAY 'SIMON SAYS GAME'

    RULES :
  • Click on start to start the game.
  • One of the four buttons will blink (color changes to white for a very short moment).
    You need to remember which button blinked and click on it to move to next level.
  • If you click the right button you will be promoted to the next level.
  • When you're promoted to the next level a new button will blink.
  • You need to remember the order of buttons blinked click on both the buttons in that same order.
  • For example, if on clicking start 'green' button blinks then in level 1 you need to click only on 'green' button. If in the next level i.e at level 2 'blue' button blinks then you need to click on 'green' button first and then 'blue' button. In next level if 'red' blinks then you need to click on 'green', 'blue' and 'red' respectively. The game continues the same way for higher levels too.
  • If you click all the buttons in correct order in a particular level then only you can play the next level.
  • You can see that no. of buttons you need to click in a particular level is equal to that level no.
  • If you click any wrong button the game stops in that moment.
  • Your score is number of levels you have successfully completed in that round.
    REMEMBER:
  • To start a new round click on 'RESTART'.
  • Don't refresh the page if you want to see scores of the rounds you played continuously.
  • You can play as many rounds as you want.
  • To know scores of all the rounds you have played click on 'Show Score Card' button.
  • You can also see the highest score among all the rounds played.
